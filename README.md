# Quiz Application with API

This project is an interactive quiz application built using HTML, CSS, and JavaScript. The quiz fetches questions from an external API and provides a timed, engaging quiz experience with real-time progress tracking.

## Features
- Questions loaded dynamically from an API
- Countdown timer for each question
- Progress bar to show quiz completion
- Multiple-choice options
- Score calculation and result display
- Restart quiz functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Quiz API

## How It Works
The application fetches quiz questions from an API, displays them one by one, tracks user selections, updates progress and time, and calculates the final score when the quiz is completed.

## Purpose
This project was developed to practice API integration, asynchronous JavaScript, and building interactive frontend applications.
